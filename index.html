This is my portfolio of data visualizations, created as part of my LSE course: PP434 - Automated Data Visualization taught by Professor Richard Davies. Every week, we were assigned coding challenges to develop our skills in fetching and scraping data, data cleaning and storage, and visualization and interactivity.

Week 1: Hosting

Task: Setting up our own Github accounts and embedding charts using the vegaEmbed function.

Task: Creating charts using the Economics Observatory Data Hub "create" tool and embedding them to our site.
